# Awesome Library
A library web page mock-up project built as an assignment for The Odin Project course to practice working JavaScript Objects.

## Features
- Usage of JavaScript Objects
- Library Management System Simulation
- Book Statistics
- Responsive design

## Technologies
- HTML
- CSS
- JavaScript
